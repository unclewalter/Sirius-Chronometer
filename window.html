Usage

On the mobile device, type:

http://<your computer's ip address>:3000

into the browser and they will see the chronometer.

To find your IP address, hold down option and click the wi-fi icon and you will the IP address listed. So if it says for instance that your IP address is 192.168.1.10. Then you would type into the mobile device browser:

http://192.168.1.10:3000
